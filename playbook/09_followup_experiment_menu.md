# Follow-up Experiment Menu

Every enriched program must be tested with at least:
- One **functional perturbation**
- One **orthogonal readout**

---

## Core Programs

### mTOR / Growth
- Perturb: Rapamycin, Torin
- Readouts: pS6, p4EBP1, SUnSET, cell size
- Control: vehicle, nutrient deprivation

### IFN / Antiviral
- Perturb: JAK inhibitor, IFN-β
- Readouts: ISG qPCR, pSTAT1
- Control: mock treated cells

### NFκB / Inflammation
- Perturb: IKK inhibitors
- Readouts: TNF, IL-1β, CXCLs, RELA localization

### ECM / Fibrosis
- Perturb: TGF-β receptor inhibitors
- Readouts: Collagen I/III IF, hydroxyproline
- Control: fibroblast fraction

### DDR / Apoptosis
- Perturb: Caspase inhibitors, DNA damage agents
- Readouts: γH2AX, Annexin V, cell survival

### Mitochondria
- Perturb: Oligomycin, FCCP
- Readouts: OCR/ECAR, ATP, ROS, membrane potential

---

## Rule

If inhibiting the pathway removes the phenotype, it is likely causal.
If not, it is likely reactive or a confounder.
